## Configuration paths

| Path | Type | Default | Description |
| --- | --- | --- | --- |
| `default_environment` | string | - | Default workspace or folder environment to select when lazytf starts. |
| `history` | object | - | History storage and retention settings. |
| `history.compression_threshold` | integer | 65536 | Compress stored output larger than this many bytes. |
| `history.enabled` | boolean | true | Enable persistent operation history. |

| `history.path` | string | - | Path to the history database file. |
| `mouse` | boolean | - | Enable mouse navigation in the UI. By default lazytf enables mouse outside tmux and disables it inside tmux to respect tmux mouse settings. Set this explicitly to override that behavior. |
| `notification` | boolean | false | Enable user notifications for important UI events. Set this explicitly to override the default behavior. |

| `presets[].name` | string | - | Preset name used with --preset. |
| `presets[].theme` | string (enum) | - | Built-in UI theme to apply when this preset is selected. |
| `presets[].workdir` | string | - | Working directory to use when this preset is selected. |

| `terraform.timeout` | string | 10m0s | Maximum time allowed for Terraform commands before lazytf cancels them. |
| `terraform.working_dir` | string | - | Default working directory used when no folder or preset overrides it. |
| `theme` | object | - | Theme settings for the lazytf UI. |
| `theme.name` | string (enum) | default | Built-in UI theme name. |
| `warnings` | object | - | Controls warning visibility in the UI and runtime diagnostics. |
| `warnings.suppress_all` | boolean | false | Suppress all non-blocking warnings shown by lazytf. |
